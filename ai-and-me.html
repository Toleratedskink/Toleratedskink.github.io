Overview

A comprehensive senior project implementing a Convolutional Neural Network (CNN) for object fault detection using a Raspberry Pi 5 (16GB) equipped with a HAILO8 AI acceleration hat delivering 26 TOPS (Tera Operations Per Second). This project demonstrates the integration of edge AI computing, computer vision, and embedded systems for real-time industrial quality control applications.

Applications

This system is designed for industrial quality control applications, particularly in manufacturing environments where real-time fault detection is critical. The edge AI approach enables deployment directly on production lines without requiring cloud connectivity, providing low-latency, privacy-preserving fault detection capabilities.
